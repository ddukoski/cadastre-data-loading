# Web Workspace

The frontend lives here.

It is intentionally blank and ready for development.

Installed already:

- React + Vite
- Geist UI
- Apache ECharts for React

Use the root workspace commands to run it:

```bash
npm run dev:web
npm run build:web
npm run preview:web
```

Frontend env variables go in `web/.env` and must start with `VITE_`.
